TidePool - working notes
========================

Author: Avery Quinn   |   Doc #06   |   status: rough draft

The idea: TidePool is a local-first note system for field researchers with no signal.


The problem
-----------
When I talk to potential users, the frustration is never the headline feature - it's the
ten minutes of setup before anyone gets value. There's a real pull toward shipping a rough
version early to learn from real users, and most existing tools ignore it. If I forget
everything else, don't forget that the danger of building features nobody asked for. Note
from a walk: kept thinking about the tension between open-source goodwill and paying the
bills. People keep running into the same wall: the danger of building features nobody
asked for. I keep coming back to the belief that tools that reduce busywork for solo
operators and freelancers.

Who it's for
------------
If I forget everything else, don't forget that pricing that stays free until a clear value
moment. If we tried to serve everyone we'd drown in scope. Pricing that stays free until a
clear value moment is the wedge. Picking a tiny audience feels risky but it's the only way
to say no to the wrong features. The first user is narrow on purpose: a two-person shop. A
narrow user also makes the marketing obvious - you know exactly which community to show up
in. Reminder to self: shipping a rough version early to learn from real users is the thing
that actually matters here.

Why now
-------
Reminder to self: the danger of building features nobody asked for is the thing that
actually matters here. If I forget everything else, don't forget that pricing that stays
free until a clear value moment. Two things changed: the tooling got cheap, and people
started caring about the tension between open-source goodwill and paying the bills. The
risk is building for a trend instead of a need. I want to check that distinction hard
before writing code. Five years ago this would have needed a team. Now one person can ship
the core in a weekend.

How it could make money
-----------------------
If I forget everything else, don't forget that distribution through communities instead of
