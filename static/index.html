Connect Google Drive (web, one-time)

About 5 minutes. You only do this once; after that, paste any Drive folder link and analyze.

  1. Open the Google Cloud Console and create a new project (any name), then select it at the top.
  2. APIs & Services → Enable APIs and services: search Google Drive API (the first result), open it, click Enable.
  3. APIs & Services → OAuth consent screen → Get started: enter an app name + your email; Audience: External; add contact email; Create.
  4. On the Audience page, leave status as Testing and under Test users → Add users add your own Gmail.
  5. Clients → Create client, type Desktop app, Create, then Download JSON.
  6. Rename it to exactly credentials.json and drop it into the app’s credentials folder:
Then tick Google Drive folder on the Analyze screen, paste a folder link, and run. A browser opens once to authorize read-only access (you’ll see an “unverified app” notice, click Advanced, then Go to Sonario; that’s normal for your own app). The token is cached locally and reused.

What should I summarize?

A file (Word, PDF, EPUB, text), a YouTube video, or a web page. YouTube videos need captions to transcribe (most do) — videos with captions disabled can't be summarized.
